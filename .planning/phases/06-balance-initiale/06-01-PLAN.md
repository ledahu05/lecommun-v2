  <files>lib/db/queries/mois.ts, app/(app)/actions.ts</files>
  <action>
1. In `lib/db/queries/mois.ts`, add two exported functions:

   a) `hasPreviousMois(annee: number, moisNum: number): Promise<boolean>` -- checks if the calendar-previous month exists in the `mois` table. Use the same month-1/year-rollover logic already in `computeBalanceReportee` (lines 83-84). Query `db.select({ id: mois.id }).from(mois).where(and(eq(mois.annee, prevAnnee), eq(mois.mois, prevMoisNum))).limit(1)` and return `rows.length > 0`.

   b) `updateBalanceReportee(id: number, value: number): Promise<void>` -- updates `balance_reportee` on the `mois` row: `db.update(mois).set({ balance_reportee: value }).where(eq(mois.id, id))`.

   - Call `revalidatePath('/')` after update
   - Follow the exact pattern from `app/(app)/depenses/actions.ts` for error handling and return type
