L'approche: BalanceCard est un RSC. Les modales DepenseForm et AjustementForm sont des Client Components. On ne peut pas les rendre DANS BalanceCard directement car BalanceCard est un RSC qui ne peut pas importer de Client Components interactifs avec state.

Solution: Ajouter les composants modaux directement dans `app/(app)/page.tsx` (qui est un RSC), en dessous ou autour de BalanceCard. L'important est l'UX -- les boutons quick-add doivent etre visuellement proches de la BalanceCard.

1. Modifier `app/(app)/page.tsx`:
   - Importer DepenseForm et AjustementForm
   - Ajouter une section quick-add SOUS la BalanceCard, avec deux boutons cote a cote dans un `div.grid.grid-cols-2.gap-3`:
     - `<DepenseForm />` (le trigger est deja dans le composant)
     - `<AjustementForm />` (le trigger est deja dans le composant)
   - Probleme: les composants DepenseForm et AjustementForm ont leur propre texte de trigger ("Ajouter une depense", "Ajouter un ajustement"). Pour le dashboard, on veut un texte plus court.
   - Solution: Ajouter une prop optionnelle `triggerLabel?: string` aux deux composants modaux. Sur le dashboard, passer `triggerLabel="Depense"` et `triggerLabel="Ajustement"`. Si non fourni, utiliser le label par defaut. Aussi ajouter une prop `triggerVariant?: "default" | "outline"` pour permettre de styliser differemment sur le dashboard (variant="outline").
   - Les boutons quick-add doivent avoir les data-testid `data-testid="quick-add-depense"` et `data-testid="quick-add-ajustement"` sur les DialogTrigger du dashboard. Pour ca, ajouter aussi une prop `triggerTestId?: string` aux composants modaux.

2. Modifier `components/depenses/DepenseForm.tsx`:
   - Ajouter les props `triggerLabel?: string`, `triggerVariant?: "default" | "outline"`, `triggerTestId?: string`
   - Les appliquer au bouton DialogTrigger

3. Modifier `components/ajustements/AjustementForm.tsx`:
   - Meme ajout de props que DepenseForm

4. Dans `app/(app)/page.tsx`, les appels seront:
   ```tsx
   <div className="grid grid-cols-2 gap-3">
     <DepenseForm triggerLabel="+ Depense" triggerVariant="outline" triggerTestId="quick-add-depense" />
     <AjustementForm triggerLabel="+ Ajustement" triggerVariant="outline" triggerTestId="quick-add-ajustement" />
   </div>
   ```
   Placer cette div juste apres le BalanceSynthese, avant la fermeture du main.
